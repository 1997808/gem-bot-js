# gem-bot-js
Gem bot JS

## fork this repo

## Clone Repo 

```bash
git clone https://github.com/<githubusername>/gem-bot-js
```

## create your branch

## Implement your bot

Implement your logic, please see `scripts/main.js`

## Run your bot

Install serve

```
npm i -g serve
```

Serve the bot

```
serve .
```

access the bot

```
http://localhost:3000?username=<yourusername>
```

inspect your bot

http://172.16.100.112:5555/
